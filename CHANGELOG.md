# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added
- MCP server with `create_presentation`, `get_format_guide`, `list_themes`, `search_stock_photos`, and `generate_ai_image` tools
- HTML output with keyboard navigation, touch/swipe, fullscreen, and print-to-PDF
- PPTX output via pptxgenjs
- 4 built-in themes: default, dark, minimal, corporate
- SVG bar, pie, and line charts
- SVG radial mind maps
- Stock photo search via Pixabay API
- AI image generation via Google Gemini
- Markdown-like input format with frontmatter, slide separators, and rich content blocks
- Interactive installer supporting Claude Code, Gemini CLI, and Cursor
